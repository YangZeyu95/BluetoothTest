# BluetoothTest
Contral a serve motor via bluetooth and position senser. 
In this demo you can get the position information of the iPhone and send it to a bluetooth device named BT05. It can also receive information
from the device.
This demo is writen to control a serve motor which was controled by a Freescale K60 MCU and this system is my graduate project. 
Codes included in this demo was the most basic part to complite those function. So you can essily change it to applicate in your app.
